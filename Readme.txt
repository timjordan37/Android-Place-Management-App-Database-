An Android application that demonstrates the management of different 
locations via database

Unzip the file and load the android studio project.

The database is initialized in the r.raw folder as a file called
locations.db.

Build the app and run it on Nexus 5 API 28.

The app should start showing the list of pre loaded places from the
locations.db file.

To add a new location, hit the plus button on the top right portion 
of the screen.

To perform calculations hit the calculator icon on the top right 
portion of the screen.

To delete a place, select the place you want to delete and then
hit the trash can icon on the top right portion of the screen.

To edit a place, select the place you want to edit and then hit 
the pencil icon on the top right portion of the screen.
